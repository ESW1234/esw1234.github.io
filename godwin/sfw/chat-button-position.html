embeddedservice_bootstrap.settings.chatButtonPosition

Repositions both the floating chat button and the minimized chat frame in ECv2. Value format is "<bottom>, <right>" — distance from the bottom and right edges of the viewport. The value below is written to chatButtonPosition before init(); pick a preset (or type one) and the page reloads with it applied.
