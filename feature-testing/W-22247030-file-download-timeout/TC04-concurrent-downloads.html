Functional Correctness Manual

TC04: Back-to-Back Download Clicks Do Not Abort Each Other

Pre-conditions

  • Active session with the agent
  • Agent has sent at least two file attachments (e.g. two 2MB PDFs)
  • DevTools Network tab open

Steps to Reproduce

  1. Open this page and start a chat session.
  2. Ask the agent to send two file attachments.
  3. Once both attachments are visible in the chat, click download on the first file.
  4. Within ~200ms (immediately after), click download on the second file before the first finishes.
  5. Wait for both to commit.
  6. Check the OS download folder for both files.
  7. Check DevTools Network panel for any (canceled) requests.

Expected Results

  • Both files arrive in the OS download folder.
  • Neither request shows as (canceled) in DevTools Network.
  • No DOM iframe leak after both complete (check with document.querySelectorAll('iframe').length).
  • No console errors.
