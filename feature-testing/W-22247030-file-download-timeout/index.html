W-22247030: File Downloads Aborted by 1000ms Iframe Cleanup Timeout

Bug Summary: When an end user clicks the download button on an agent-sent file in Enhanced Messaging, the bootstrap layer creates a hidden iframe and unconditionally removes it after 1000ms. If the response hasn't committed by then, the download silently fails with net::ERR_ABORTED.
