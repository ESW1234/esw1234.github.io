Iframe Lifecycle & DOM Hygiene Manual

TC03: 50 Sequential Downloads Do Not Leak DOM Nodes

Pre-conditions

  • Active session with the agent
  • Agent able to send multiple small attachments (~100KB each)
  • DevTools Console open

Steps to Reproduce

  1. Open this page and start a chat session.
  2. Before any downloads, run this in the DevTools Console to record baseline iframe count:
// Record baseline console.log('Baseline iframes:', document.querySelectorAll('iframe').length);
  1. Have the agent send a small file attachment.
  2. Click download on the attachment. Wait for it to complete.
  3. Repeat steps 3–4 at least 50 times (or as many as practical).
  4. After all downloads complete, run this in the Console:
// Check for orphan iframes after downloads console.log('Final iframes:', document.querySelectorAll('iframe').length); console.log('Orphan download iframes:', document.querySelectorAll('iframe[style*="display: none"]').length);
  1. Take a heap snapshot (DevTools → Memory → Take snapshot) and note the total size.

Expected Results

  • Final iframe count equals baseline (no orphan download iframes remain).
  • All files downloaded successfully.
  • Heap snapshot delta < 5MB compared to baseline.
  • No removeChild errors in the Console.
