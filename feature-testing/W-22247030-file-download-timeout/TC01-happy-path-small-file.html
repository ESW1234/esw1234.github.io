Functional Correctness Manual

TC01: Same-tab Download of a Small PDF Completes Under Normal Network

Pre-conditions

  • Bootstrap loaded with default settings (shouldOpenFileInNewTab=false)
  • Active live-agent session in MIAW Enhanced Messaging
  • Agent has just sent a small PDF (<100KB)

Steps to Reproduce

  1. Open this page and wait for the chat widget to appear (bottom-right).
  2. Click the chat FAB to open the conversation.
  3. Start a conversation with the agent. Ask it to send you a file.
  4. When the agent sends a file attachment, click the download button on the attachment.
  5. Wait for the browser to commit the download.
  6. Inspect the OS download folder for the file.
  7. Open the file and verify the content matches the source.

Expected Results

  • File downloads on first click without retry.
  • File name and content match the source byte-for-byte.
  • DevTools Network tab shows the iframe request reaches Status 200/finished — never (canceled).
  • Console is free of errors and warnings.
